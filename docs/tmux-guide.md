# 🧩 tmux 가이드

원격 개발 환경(WSL + tmux)에서 tmux를 처음 쓰는 사람을 위한 가이드.
**개념 → 실행 명령 → 단축키** 순서로 정리.

> 내 셋업 기준: Termius(SSH)로 접속하면 자동으로 `work` 세션에 진입.

---

## 1. 개념

tmux = **WSL 안에서 계속 돌아가는 작업 공간(책상).** 내가 보고 있든 말든 안 꺼진다.

### 비유 — "책상"

- **attach(붙기)** = 책상에 **앉기**
- **detach(나가기)** = **일어나 자리 뜨기** → 책상은 **그대로 유지**
- 접속하면 자동으로 `work` 책상에 앉음. 다시 앉으면 떠날 때 그 모습 그대로.
- 핵심: 내가 안 보고 있어도 **책상 위 작업(claude·dev 서버…)은 계속 진행**.

### 계층 구조

```
방 (tmux 서버, WSL 안)
 └─ 책상 (세션 session)      # "work"가 기본. 여러 개 가능
     └─ 탭 (윈도우 window)
         └─ 칸 (분할 pane)
```

### 나갔다 들어오기

| 상황 | 작업은? |
|---|---|
| `Ctrl-b d`(detach) / 그냥 앱 닫기 / 네트워크 끊김 | 전부 **유지** ✅ — 재접속하면 그대로 복귀 |
| **PC 재부팅** | **사라짐** ❌ — 서버 자체가 내려감 (빈 work 새로 생성) |

> 한 줄 정리: tmux는 **"끊김"엔 강하지만 "재부팅"은 못 버팀.** 끊겨도 작업은 PC에서 계속 돈다.

## 2. tmux 실행 명령

셸 프롬프트에서 치는 `tmux ...` 명령. (tmux **밖**에서, 또는 세션을 통째로 다룰 때)

### 세션 시작 / 붙기

| 명령 | 설명 |
|---|---|
| `tmux` | 새 세션 시작 (이름 자동: 0, 1, 2…) |
| `tmux new -s work` | **이름 지정** 새 세션 (`-s` = session name) |
| `tmux new -s work -d` | 만들되 **안 들어감** (백그라운드, `-d` = detached) |
| `tmux new -A -s work` | **있으면 붙고, 없으면 만듦** (`-A`) — 안전해서 자주 씀 |
| `tmux new -s work 'npm run dev'` | 세션 시작하며 **명령 바로 실행** |
| `tmux a` (= `attach`) | 마지막 세션에 **붙기** |
| `tmux a -t work` | 특정 세션 붙기 (`-t` = target) |
| `tmux a -d -t work` | 붙으며 **다른 접속 떼어냄** (`-d`) — 화면 깨질 때 유용 |

### 목록 / 확인

| 명령 | 설명 |
|---|---|
| `tmux ls` (= `list-sessions`) | 세션 목록 |
| `tmux lsw` | 현재 세션의 윈도우 목록 |
| `tmux has -t work` | 세션 존재 확인 (스크립트용) |

### 이름 변경 / 종료

| 명령 | 설명 |
|---|---|
| `tmux rename-session -t work dev` | 세션 이름 변경 |
| `tmux kill-session -t work` | 특정 세션 종료 |
| `tmux kill-server` | ⚠️ **모든 세션 종료** (tmux 통째로) |

### 기타

| 명령 | 설명 |
|---|---|
| `tmux source ~/.tmux.conf` | 설정 파일 다시 읽기 |
| `tmux -V` | 버전 확인 |

> ⚠️ tmux **안에서** `tmux new` 금지 — 이미 tmux 안인데 또 치면 "nested" 경고(중첩). 안에서 새 세션은 `Ctrl-b :` → `new -s 이름`.

## 3. `Ctrl-b` 단축키

tmux **안에서**의 명령은 거의 다: **`Ctrl-b` → (떼고) → 키**
(📱 Termius: 툴바 `Ctrl` 탭 → `b` → 키)

### 나가기 / 돌아오기

| 키 | 동작 |
|---|---|
| `Ctrl-b` `d` | 나가기(detach) — 작업 유지 |
| (그냥 앱 닫기) | 제일 흔한 나가기 (자동 복귀) |

### 윈도우 (탭)

| 키 | 동작 |
|---|---|
| `Ctrl-b` `c` | 새 윈도우 |
| `Ctrl-b` `n` / `p` | 다음 / 이전 |
| `Ctrl-b` `0`~`9` | 번호로 점프 |
| `Ctrl-b` `,` | 이름 변경 |
| `Ctrl-b` `w` | 목록에서 선택 |
| `Ctrl-b` `&` (또는 `exit`) | 윈도우 닫기 |

### 분할 (pane)

| 키 | 동작 |
|---|---|
| `Ctrl-b` `%` | 좌우 분할 |
| `Ctrl-b` `"` | 상하 분할 |
| `Ctrl-b` 화살표 | pane 이동 |
| `Ctrl-b` `z` | 확대/축소 토글 |
| `Ctrl-b` `x` (또는 `exit`) | pane 닫기 |

### 세션 (책상)

| 키 | 동작 |
|---|---|
| `Ctrl-b` `:` → `new -s 이름` | 새 세션 생성+이동 |
| `Ctrl-b` `s` | 세션 목록 / 전환 |
| `Ctrl-b` `)` / `(` | 다음 / 이전 세션 |
| `Ctrl-b` `$` | 세션 이름 변경 |
| `Ctrl-b` `D` | 특정 클라이언트 detach (대문자) |

### 스크롤 / 복사

| 키 | 동작 |
|---|---|
| 손가락 드래그 | 스크롤 (mouse on) |
| `Ctrl-b` `[` | 스크롤 모드 진입 |
| `q` | **스크롤 모드 나가기** |
| 길게 눌러 선택 | 복사 (Termius → 클립보드) |

### 기타

| 키 | 동작 |
|---|---|
| `Ctrl-b` `?` | 단축키 전체 도움말 (`q`로 나감) |
| `Ctrl-b` `:` | tmux 명령 직접 입력 |
| `Ctrl-b` `t` | 큰 시계 |

> **입력이 갑자기 안 먹히면** → 스크롤 모드에 갇힌 것: `q`. / 어떤 명령에 갇혔으면: `Ctrl-c`. / `Ctrl-b`가 안 먹으면 글자가 셸로 새니(예: `dd` 사고) Ctrl이 눌렸는지 확인.

---

*자주 쓰는 건 결국 `Ctrl-b d`/`c`/`n` 몇 개. 막히면 `Ctrl-b ?`.*
