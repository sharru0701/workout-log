# Data backfills (1회성 운영 데이터 보정 기록)

이 디렉터리는 **프로덕션 DB에 1회 수동 실행한 데이터 보정 SQL을 감사/재현 목적으로 보관**한다.

- 여기 있는 SQL은 **자동 실행되지 않는다.** Drizzle 마이그레이션(`src/server/db/migrations`)과 달리
  journal에 등록되지 않으며 `db:migrate`가 실행하지 않는다.
- 스키마 변경이 아니라 **특정 사용자/기간의 데이터 정합성 보정**이라, 모든 환경에서 재실행하면 안 되는
  값(가정한 체중 등)을 포함한다. 따라서 마이그레이션이 아닌 기록물로만 둔다.
- 각 파일 상단 주석에 **배경 · 적용 대상 · 실행 환경 · 실행 일자**를 남긴다.
- 실행 전 반드시 동봉된 미리보기(SELECT)로 영향 행을 확인하고, 가능하면 백업 후 적용한다.
