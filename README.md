# Workout Log

이 저장소는 현재 `web` 앱 중심으로 개발합니다.

빠른 시작:

```bash
git clone <repo-url>
cd workout-log/web
pnpm install
pnpm dev
```

배포는 Vercel(웹앱) + Supabase(Postgres)로 운영합니다.
